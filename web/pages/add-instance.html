How manual setup works

Dockpal will register this instance in an enrolling state. We will provide a pre-configured docker run command. You simply copy and execute that command on your remote server's terminal to deploy and link the agent.

Dockpal Connection Modes

🌐 Direct Mode

In direct mode, the Dockpal server acts as the client and initiates API requests to the agent running on port 9273. Ensure that port is open on your firewall and the agent host is reachable via VPN or public IP.

☁️ Edge Mode (Recommended)

In edge mode, the remote agent connects directly to the Dockpal server using a secure persistent WebSocket connection. No inbound firewall rules are needed on the remote agent server, making this highly secure and firewall friendly.

🛡️ Security & Cryptography

All SSH passwords and private keys are encrypted at rest using high-performance AES-256-GCM. The unique cryptographic encryption key is securely derived on startup using cryptographic secrets unique to this Dockpal instance. Your credentials are never stored in plaintext and never leave this server.
